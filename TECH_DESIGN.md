# AI 海龟汤游戏 - 技术设计文档 (TECH_DESIGN)

## 1. 建造材料 (技术栈选择)
- **前端页面 (客人看到的餐厅大堂)**：使用 React + TypeScript + Vite 搭建，速度快、好维护。
- **装修刷漆 (排版和颜色样式)**：使用 Tailwind CSS，方便快速调出深蓝色悬疑风。
- **后台控制室 (处理秘密数据的地方)**：使用 Node.js + Express 搭建简单的后端。
- **AI 大脑 (聪明的主持人)**：接入大模型 API（比如 DeepSeek），让它来当法官。

## 2. 餐厅房间划分 (核心页面路由)
- `/` (首页)：也就是 **Home 页面**，展示游戏大厅和故事列表。
- `/game/:id` (游戏页)：也就是 **Game 页面**，客人和 AI 在这里一问一答推理。
- `/result` (结算页)：也就是 **Result 页面**，真相大白的地方，给客人看完整的汤底。

## 3. 核心食材清单 (数据模型设计)
每个海龟汤故事（Story）在数据里必须长这样：
- `id`: 故事的唯一编号
- `title`: 故事标题
- `difficulty`: 难度（简单/中等/困难）
- `surface`: 汤面（一开始展示给客人看的离奇片段）
- `bottom`: 汤底（绝对不能提前泄露的完整真相）
